Senior Thesis - Early Rise Supernova Light Curve Model Fitting with TESS Data

  • August 2020 -- June 2021. Advisors: Richard French (Wellesley), Tansu Daylan (MIT)
  • Wrote a Python data mining program to identify known Type Ia supernovae observed by TESS. (See here)
  • Performed detailed follow up including model fitting using emcee for comparison against the fireball model.
  • thesiscode

Project is continuing (June 2021-present) as the development of an automated pipeline to identify and model fit to TESS observed SN. etsfit uses Bayesian analysis to fit power law models with complex backgrounds based on the TESS pointing model, the light curve extractor, and/or a Matern-3/2 Gaussian Process kernal to optimize the fit. We hope to publish as Gordon+ 2022.
