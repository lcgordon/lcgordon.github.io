Device Parallelized Optimization for FLAG

  • Part of the summer 2023 LANL Parallel Computing Summer School
  • Initial work on GPU offloading analysis for FLAG multi-physics code. OpenMP and CUDA primarily, with a focus on changes required for the build system to compile with new directives in place.

Early Time Supernova FITter (etsfit)

  • August 2020 -- June 2021. Advisors: Richard French (Wellesley), Tansu Daylan (MIT)
  • Wrote a Python data mining program to identify known Type Ia supernovae observed by TESS.
  • etsfit uses Bayesian analysis for model fitting. Includes options for complex backgrounds based on Gaussian Processes and TESS pointing data and a streamlined pipeline for user-inserted likelihood functions. thesiscode

TESS Exoplanet Follow Up Observing Program

  • January 2020 -- June 2021. Advisor: Kim McLeod (Wellesley)
  • Observing TESS candidate planets using the local 0.7m PlaneWave and performing data reduction.
  • Assisting with target scheduling, training new hires, and observational projects for the astronomy research methods course.
  • Nth author on TESS Delivers Five New Hot Giant Planets Orbiting Bright Stars from the Full Frame Images for work on confirmation of TOI 628-b.

A Compact Mult-Beam Linear Accelerator Prototype

  • August 2019 -- December 2019. Advisor: Arun Persaud (LBNL)
  • SULI Internship in the Accelerator Technology and Applied Physics Dept.
  • Electrical engineering work on parts testing for new components (RF voltage amplifier, microelectromech. wafers) for an energy upgrade to a prototype accelerator design.
  • Computational physics work on updating Python simulations of the internal fields and ion motion within the accelerator.

Searching for Dual Quasars in Archival Hubble Data

  • June 2019 -- August 2019. Advisor: Eilat Glikman (Middlebury College)
  • Wrote a Python search algorithm to find candidate double quasar systems in the Hubble archive using contour maps. Analyzed resulting density of identified candidates.
  • Code has since been reworked into a small Python package that will allow users to produce contour plots for any Hubble observed target. Lindsey Gordon
